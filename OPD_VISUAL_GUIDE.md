## Color Codes & Tags

### Diagnosis Tags
- `[Chính]` - Red background - Primary diagnosis
- `[Phụ]` - Blue background - Secondary diagnosis

### Queue Status Tags
- `[Chờ khám]` - Orange - Waiting
- `[Đang khám]` - Blue - In examination
- `[Hoàn thành]` - Green - Completed

### Order Type Tags
- `[XN]` - Laboratory test
- `[CĐHA]` - Imaging
- `[TT]` - Procedure
- `[Thuốc]` - Medication
- `[DV]` - Service

### Patient Type Tags
- `[BHYT]` - Green - Insurance
- `[Viện phí]` - Blue - Self-pay
- `[Dịch vụ]` - Purple - Service

## User Interactions

### Click Actions
1. **Queue Row** → Select patient and load info
2. **History Button** → Open history modal
3. **Save Button** → Save as draft
4. **Complete Button** → Confirm and complete
5. **Print Button** → Generate report
6. **Add Diagnosis** → Add to diagnosis table
7. **Add Order** → Add to order table
8. **Delete Row** → Remove from table

### Auto-Actions
1. **Auto-save** → Every 30 seconds (silent)
2. **BMI calculation** → When weight/height change (future)
3. **Queue refresh** → After completion

### Search/Autocomplete
1. **Patient Search** → Search as you type
2. **ICD-10 Search** → Autocomplete dropdown
3. **Service Search** → Autocomplete dropdown
